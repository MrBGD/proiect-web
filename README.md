# proiect-web (s1mple from NullCTF)

Architected and deployed a custom web application utilizing a Python/Flask backend and an HTML/CSS/JavaScript frontend to serve as a practical demonstration of web security concepts.
Successfully integrated the VirusTotal REST API, enabling the application to query and retrieve live threat intelligence data URL scanning, demonstrating proficiency in handling external APIs and JSON data parsing.
Purposely engineered the backend to include critical web vulnerabilities, specifically SQL Injection (SQLi) and Server-Side Template Injection (SSTI).
Designed the application for demonstrating how these vulnerabilities are exploited in the wild and highlighting the importance of secure coding practices.
